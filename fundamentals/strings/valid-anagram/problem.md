# Valid Anagram

## English
Given two strings `s` and `t`, return `true` if `t` is an anagram of `s`, otherwise return `false`.

Implement `solve(data)` in `python/solution.py`, where `data` is a dictionary with:
- `s`: source string
- `t`: candidate anagram

### Example
- Input: `{"s": "anagram", "t": "nagaram"}`
- Output: `True`

## Espanol
