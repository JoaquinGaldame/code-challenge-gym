# Longest Substring Without Repeating Characters

## English
Given a string `s`, find the length of the longest substring without repeating characters.

Implement `solve(data)` in `python/solution.py`, where `data` is a string.

### Example
- Input: `"abcabcbb"`
- Output: `3`

## Espanol
Dada una cadena `s`, encuentra la longitud de la subcadena mas larga sin caracteres repetidos.

Implementa `solve(data)` en `python/solution.py`, donde `data` es una cadena.
