# Best Time To Buy And Sell Stock

## English
You are given a list `prices` where `prices[i]` is the price of a stock on day `i`.

Choose a single day to buy one stock and a different future day to sell it. Return the maximum profit you can achieve. If no profit is possible, return `0`.

Implement `solve(data)` in `python/solution.py`, where `data` is a list of integers.

### Example
- Input: `[7, 1, 5, 3, 6, 4]`
- Output: `5`

## Espanol
Se te da una lista `prices` donde `prices[i]` es el precio de una accion en el dia `i`.

Elige un unico dia para comprar una accion y un dia futuro distinto para venderla. Devuelve la ganancia maxima posible. Si no se puede obtener ganancia, devuelve `0`.

Implementa `solve(data)` en `python/solution.py`, donde `data` es una lista de enteros.
