# Valid Parentheses

## English
Given a string `s` containing only the characters `()[]{}`, determine whether the input string is valid.

A string is valid if:
- open brackets are closed by the same type of brackets
- open brackets are closed in the correct order
- every closing bracket has a corresponding opening bracket

Implement `solve(data)` in `python/solution.py`, where `data` is a string.

### Example
- Input: `"()[]{}"`
- Output: `True`

## Espanol
Dada una cadena `s` que contiene solo los caracteres `()[]{}`, determina si la cadena es valida.

Una cadena es valida si:
- cada parentesis de apertura se cierra con el mismo tipo
- los cierres ocurren en el orden correcto
- cada cierre tiene su apertura correspondiente

Implementa `solve(data)` en `python/solution.py`, donde `data` es una cadena.
