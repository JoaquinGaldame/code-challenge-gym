# Climbing Stairs

## English
You are climbing a staircase. It takes `n` steps to reach the top.

Each time you can climb either `1` or `2` steps. Return how many distinct ways you can reach the top.

Implement `solve(data)` in `python/solution.py`, where `data` is an integer `n`.

### Example
- Input: `4`
- Output: `5`

## Espanol
Estas subiendo una escalera. Se necesitan `n` escalones para llegar arriba.

Cada vez puedes subir `1` o `2` escalones. Devuelve cuantas formas distintas hay de llegar a la cima.

Implementa `solve(data)` en `python/solution.py`, donde `data` es un entero `n`.
