# Coin Change

## English
You are given an integer array `coins` representing coin denominations and an integer `amount` representing a total amount of money.

Return the fewest number of coins needed to make up that amount. If the amount cannot be formed, return `-1`.

Implement `solve(data)` in `python/solution.py`, where `data` is a dictionary with:
- `coins`: list of positive integers
- `amount`: non-negative integer

### Example
- Input: `{"coins": [1, 2, 5], "amount": 11}`
- Output: `3`
