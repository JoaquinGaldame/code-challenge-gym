# Fundamentals

Esta carpeta organiza los ejercicios por patron clasico de entrevista.

## Objetivo
Antes de saltar a los tracks por nivel, aqui puedes practicar la idea central detras de cada tipo de problema.

## Patrones incluidos
- `arrays`
- `strings`
- `hashmaps`
- `stacks`
- `queues`
- `recursion`
- `trees`
- `graphs`
- `dynamic-programming`

## Como usar esta carpeta
1. Lee el `problem.md` del ejercicio.
2. Resuelve en `python/starter.py` o `javascript/starter.js`.
3. Ejecuta el test del starter.
4. Cuando termines, revisa `solution.py` o `solution.js`.

## Consejo
Si un patron te cuesta, no cambies de tema enseguida. Resuelve 2 o 3 ejercicios del mismo grupo hasta que la tecnica salga de memoria.
