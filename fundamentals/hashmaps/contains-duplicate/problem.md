# Contains Duplicate

## English
Given an integer array `nums`, return `true` if any value appears at least twice in the array, and return `false` if every element is distinct.

Implement `solve(data)` in `python/solution.py`, where `data` is a list of integers.

### Example
- Input: `[1, 2, 3, 1]`
- Output: `True`

## Espanol
Dado un arreglo de enteros `nums`, devuelve `true` si algun valor aparece al menos dos veces en el arreglo, y `false` si todos los elementos son distintos.

Implementa `solve(data)` en `python/solution.py`, donde `data` es una lista de enteros.
