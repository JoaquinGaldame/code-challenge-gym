# Number Of Islands

## English
Given a 2D grid of `"1"` values (land) and `"0"` values (water), count the number of islands.

An island is formed by connecting adjacent lands horizontally or vertically. You may assume all four edges of the grid are surrounded by water.

Implement `solve(data)` in `python/solution.py`, where `data` is a matrix represented as a list of lists of strings.

### Example
- Input: `[["1","1","0"],["1","0","0"],["0","0","1"]]`
- Output: `2`

## Espanol
Dada una grilla bidimensional con valores `"1"` (tierra) y `"0"` (agua), cuenta la cantidad de islas.

Una isla se forma conectando celdas de tierra adyacentes en horizontal o vertical. Puedes asumir que los cuatro bordes de la grilla estan rodeados de agua.

Implementa `solve(data)` en `python/solution.py`, donde `data` es una matriz representada como lista de listas de cadenas.
