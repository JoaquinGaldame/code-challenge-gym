# Subsets

## English
Given an array of unique integers `nums`, return all possible subsets (the power set).

The solution set must not contain duplicate subsets. You may return the subsets in any order.

Implement `solve(data)` in `python/solution.py`, where `data` is a list of integers.

### Example
- Input: `[1, 2, 3]`
- Output: `[[], [1], [2], [1, 2], [3], [1, 3], [2, 3], [1, 2, 3]]`

## Espanol
Dado un arreglo de enteros unicos `nums`, devuelve todos los subconjuntos posibles.

El conjunto resultado no debe contener subconjuntos duplicados. Puedes devolverlos en cualquier orden.

Implementa `solve(data)` en `python/solution.py`, donde `data` es una lista de enteros.
